# EXPERIMENT – 7

## Aim

Implementing a **GUI based calculator application** and drawing different **figures on a Canvas**.

---

* Implement a **GUI based calculator application**.  
  It has **two TextFields for two input numbers**, **one TextField for result**, and **four Buttons** named **Add, Sub, Mul, and Div** for **addition, subtraction, multiplication, and division** respectively.

* Write a program to **draw various figures on a Canvas**.  
  The user selects a **figure from a CheckboxGroup**, and the **selected figure is then displayed on the Canvas**.
