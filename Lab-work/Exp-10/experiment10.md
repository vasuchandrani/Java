# EXPERIMENT – 10

## Aim

Write the programs that use the concept of **Threads**.

---

## Program 1

Write an application to display **moving banner**.

- Enter **message to be moved**, **color**, and **size** through three different **TextFields**.
- Select direction (**left** or **right**) using **Choice**.
- On pressing **Start Button**, the message should start moving.

---

## Program 2

Write an **applet** that shows **four moving objects (balls)** in one 2D-box.


Four directions are used.  
On touching the **boundary** of box, the object (**ball**) changes its direction and starts moving in that changed direction.
