# EXPERIMENT – 9

## Aim

Write a program that uses the concept of **Exception Handling**.

---

* Write a program to meet the following requirements:

  * Create an **array with 100 elements** that are **randomly chosen**.

  * Create a **TextField to enter an array index** and another **TextField to display the array element** at the specified index.

  * Create a **Show Button** to display the array element.  
    If the **specified index is out of bound**, display the message **"Out of Bound"**.
