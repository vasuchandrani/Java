# EXPERIMENT – 4

## Aim

Write the programs using the concept of **Generic class, Inheritance, Interface and Package**.

---

## Program

- Write a program to **add and to multiply two matrices** using **Generic class concept**.

- Create an **abstract class `Shape`** and derived classes **`Rectangle`** and **`Circle`** from `Shape` class.  
  Implement abstract method of shape class in `Rectangle` and `Circle` class.

- `Shape` class contains:
  - `origin (x, y)` as data member
  - `display()` and `area()` as abstract methods

- `Circle` class contains:
  - `radius` as data member

- `Rectangle` class contains:
  - `length`
  - `width`

- Use **Inheritance, Overloading and Overriding** concepts.

- Write a program to show an **implementation of Package**.

- Write a program to show an **implementation of Interface**.

---

## Procedure

- To perform the program using **generic class concept**, you have to create the variable of type **Object class** in a separate class.

- To perform the operation on matrices of type **integer**, create a separate class to **convert the Object type variable into Integer type variable**.

- In the **main class**, create `main()` which will create the **objects of the above classes**.

- Provide the **logic**, then **compile and run the program**.

- To create an **abstract class**, use the concept of **inheritance**.
